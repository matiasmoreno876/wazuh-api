## Description
This project is a challenge by [Wazuh](https://wazuh.com/).

This project uses the [Hapi.js](https://hapi.dev/tutorials/gettingstarted/?lang=en_US) library.
## Installation

```bash
$ npm install
```

## Running the app

```bash
# Run
$ npm start

# development
$ npm run develop

# build app
$ npm run build
```
##### Note
This application will run on [localhost: 3000](https://wazuh.com/)

## Test

```bash
# unit tests
$ npm run test
```
